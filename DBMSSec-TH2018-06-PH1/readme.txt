1. Thư mục nộp bài gồm:
- 1 folder chứa source code ứng dụng
- 1 folder Script sql
- 1 file docx báo cáo (chứa bảng phân công công việc, đánh giá cũng như kết quả)

2. Hướng dẫn Run
- Chạy file System.sql trong thư mục Script DB & stored Proc với user System
- Chạy file U_QLBV_QUANTRIVIEN.sql trong thư mục Script DB & stored Proc với user U_QLBV_QUANTRIVIEN
- Mở netBeans và chạy file Home.java đầu tiên
